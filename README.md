
ТТТ - изначально настольная игра, которая обрела свою популярность в игре Garry's Mod. Задача у этой игры проста - предатели должны убить невинных, невинные должны прожить как можно дольше, а детектив должен расследовать убийства по невинным используя свои методы дедукции.
Данный репозиторий является русскоязычной версией оригинального репозитория SSS от Goob Station.
